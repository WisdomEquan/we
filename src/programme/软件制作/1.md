# 小六壬自动起盘小工具

<!-- more -->

![](/assets/images/小六壬.jpg)

## 介绍

使用方式主要分为2种：
1. 根据当前时间自动起盘，自动起盘直接点击起盘结果位即可。
2. 手动输入某个时间起盘，手动起盘输入阳历时间，按下回车键起盘，会自动转换为阴历时间。

## 结果解析方向

这里只说一下我个人解盘的小见地，不代表真正的正确解析方式。

结果分别有三个，以此截图为例，这里的天、地、人，是我在解盘时代入的个人分析词汇。

|      | 分析方向                                       |
| ---- | ---------------------------------------------- |
| 天   | 我将它理解为整件事外部的不可抗力因素、大环境。 |
| 地   | 表示当下你的状态                               |
| 人   | 表示这件事最后的结果                           |

在实际案例中，我一般只看最后2个结果。因为第一个结果为外部大环境，是我们无法改变的。

如果你的提问是：今天要和讨厌的人会面的吉凶。

那么从这两个结果来看，你当下的状态就是很容易会跟人发生冲突，但是你们会面最终的结果是没有问题的。

总之，小六壬只是用来预测简单问题的吉凶祸福。如果你想知道具体事情的发展方向，以及未来的可能性，那就需要更加复杂的占卜方式，比如紫占。



## 一些案例分析思路

曾经我在和一位好兄弟在探讨结果分析思路时，他提到如果第一个结果是 **吉**，而第二个结果是 **凶**，那么这种结果属于是吉中藏凶，属于大凶。他说：就像你晚上睡觉睡得很安稳，但是房间煤气泄漏了。

后来，我在实际应用中，我会更多的使用GPT根据五行来分析结果，这样我能更清楚的知道结果的相生相克，至少比你直接使用对照表分析的更加全面一点。
