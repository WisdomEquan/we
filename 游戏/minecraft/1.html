去掉七咒之戒的方法

去掉七咒之戒的方法

我将告诉你普通玩家无作弊的情况下如何去掉

简述

我在玩【工匠：技艺革新】整合包，开局会强制给你戴一个七咒之戒。

去掉的方法分为4种

  1. 拥有OP权限，直接执行/curios clear @s
  2. 没有作弊权限的单人存档，打开局域网模式并启用作弊，然后执行命令。
  3. 在服务器中请求OP、服主，删除在save/world/playerdata目录下你的UUID文件，然后重连服务器。
  4. 无作弊的单人存档，无法打开局域网模式作弊，使用数据包让普通用户也能执行去掉的命令。

本文主要说明如何在第4种情况下，去掉七咒之戒的数据包制作方法。

一、文件结构

创建文件夹：

universal_curios_clear

结构如下：

universal_curios_clear/
│
├── pack.mcmeta
│
└── data/
    ├── minecraft/
    │   └── tags/
    │       └── functions/
    │           ├── load.json
    │           └── tick.json
    │
    └── ringclear/
        ├── functions/
        │   ├── load.mcfunction
        │   ├── tick.mcfunction
        │   └── clear.mcfunction
        │
        └── advancements/
            └── trigger_clear.json

二、pack.mcmeta

文件：

pack.mcmeta

内容：

{
  "pack": {
    "pack_format": 10,
    "description": "Universal Curios Clear Datapack"
  }
}

三、load.json

路径：

data/minecraft/tags/functions/load.json

内容：

{
  "values": [
    "ringclear:load"
  ]
}

四、tick.json

路径：

data/minecraft/tags/functions/tick.json

内容：

{
  "values": [
    "ringclear:tick"
  ]
}

五、load.mcfunction

路径：

data/ringclear/functions/load.mcfunction

内容：

scoreboard objectives add clear_ring trigger
say [RingClear] Datapack Loaded

六、tick.mcfunction

路径：

data/ringclear/functions/tick.mcfunction

内容：

scoreboard players enable @a clear_ring

execute as @a[scores={clear_ring=1..}] run function ringclear:clear

scoreboard players set @a[scores={clear_ring=1..}] clear_ring 0

七、clear.mcfunction

路径：

data/ringclear/functions/clear.mcfunction

内容：

curios clear @s

say §a[RingClear] 饰品已清除

八、trigger_clear.json

路径：

data/ringclear/advancements/trigger_clear.json

内容：

{
  "criteria": {
    "requirement": {
      "trigger": "minecraft:tick"
    }
  }
}

九、安装方法

将整个文件夹：

universal_curios_clear

放入：

世界存档/datapacks/

例如：

.minecraft/saves/你的世界/datapacks/universal_curios_clear

进入世界后执行：

/reload

即可加载。

十、玩家使用方法

普通玩家直接输入：

/trigger clear_ring

即可自动执行：

curios clear @s

不需要 OP。

十一、扩展功能（可选）

你后面可以继续扩展：

1、清除指定饰品

例如：

curios clear @s charm

或者：

clear @s mod:ring

2、多命令系统

例如：

/trigger clear_ring
/trigger clear_all
/trigger clear_charm

十二、不同版本 pack_format 对照

如果换版本：

1.19.210
1.19.312
1.19.413
1.20 ~ 1.20.115
1.20.218
1.20.426
1.20.541

只需要修改：

pack.mcmeta

中的：

"pack_format"

即可。
